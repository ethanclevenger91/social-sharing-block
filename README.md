Just add \"share\" to the classes of a social links block with permalink context (such as in an FSE single post template).

## Description
As lightweight as it comes. This plugin allows you to repurpose the existing WordPress Social Links block to create share links.

This doesn't load any dependencies, no third-party scripts, no tracking. Just as light as the original block.

To use, simply add "share" to the "Additional CSS Classes" section of the parent Social Links block. Please note that _something_ has to be entered in each Social Link block's URL, but it will be overwritten for any supported services. Unsupported services will not be rendered.

**Supported Services:**
- Facebook
- Twitter
- X
- LinkedIn
